## Описание

В соответствии с дизайн-системой все интерактивные элементы должны иметь определенную обводку при фокусе, но только при переключении с клавиатуры.
При клике мышки обводки быть не должно.

Для получения такого поведения — можно воспользоваться компонентом `<KeyboardFocusable />` или хуком `useKeyboardFocusable`.

Также, стили для фокуса доступны через миксин `focus-outline`.

## Хук useKeyboardFocusable

```tsx
const ref = useRef<HTMLButtonElement>(null);
const { focused } = useKeyboardFocusable(ref);

return (
    <button ref={ref} className={cn({ [styles.focused]: focused })}>focus me</button>
);
```
