## Описание

Набор компонентов для оформления текста, реализующий [гибридную систему типографики.](https://www.figma.com/file/FWya6P55TnXkiVGaM2qBio/2020-Web-Hybrid-Typography?node-id=593%3A0)

> **_NOTE:_** Компоненты не устанавливают свойство `font-family`, кроме случаев, когда используется шрифт 'Styrene UI'. Подразумевается, что шрифт будет установлен у родителя.

## Typography.Heading

Компонент заголовка, использующий шрифт 'Styrene UI'.

## Typography.HeadingClassic

Компонент заголовка, использующий шрифт 'system-ui'.

<Story name='Typography.HeadingClassic'>

        const VIEW_TYPES = [
            'headline-classic-xlarge',
            'headline-classic-large',
            'headline-classic-normal',

## Typography.Text

Компонент для оформления текста. Позволяет задать внешний вид, тэг и цвет текста.

<Story name='Typography.Text'>
    {React.createElement(() => {
        const VIEW_TYPES = [
            'paragraph-primary-large',
            'paragraph-primary-normal',
            'paragraph-primary-small',
            'paragraph-secondary-large',
            'paragraph-secondary-normal',
            'paragraph-secondary-small',
            'paragraph-component',
            'paragraph-caps',
            'accent-primary-large',
            'accent-primary-normal',
            'accent-primary-small',
            'accent-secondary-large',
            'accent-secondary-normal',
            'accent-secondary-small',
            'accent-component',
            'accent-caps',
            'action-primary-large',
            'action-primary-normal',
            'action-primary-small',
            'action-secondary-large',
            'action-secondary-normal',
            'action-secondary-small',
            'key-xlarge',
            'key-large',
            'key-normal',
            'key-small',
            'key-xsmall',
            'legacy-styrene-normal',
            'legacy-primary-small',
        ];
        const view = select('view', VIEW_TYPES, 'paragraph-primary-large');
        const type = select('type', ['', 'secondary', 'negative', 'positive', 'attention'], '');
        const tag = select('tag', ['p', 'span'], 'p');
        const className = text('className', '');
        const dataTestId = text('dataTestId', '');
        return (
            <div style={{ fontFamily: 'var(--font-family-system)' }}>
                <Typography.Text view={view} type={type} tag={tag} className={className} dataTestId={dataTestId}>
                    Космологи́ческая сингуля́рность — состояния Вселенной в определённый момент времени в прошлом,
                    когда плотность энергии (материи) и кривизна пространства-времени были очень велики — порядка
                    планковских значений. Это состояние, вместе с последующим этапом эволюции Вселенной,
                    пока плотность энергии (материи) оставалась высокой, называют также Большим взрывом.
                    Космологическая сингулярность является одним из примеров гравитационных сингулярностей,
                    предсказываемых общей теорией относительности (ОТО) и некоторыми другими теориями гравитации.
                </Typography.Text>
                <Typography.Text view={view} type={type} tag={tag} className={className} dataTestId={dataTestId}>
                    Проблема существования космологической сингулярности является одной из наиболее серьёзных
                    проблем физической космологии. Дело в том, что никакие наши сведения о том, что произошло
                    после Большого взрыва, не могут дать нам никакой информации о том, что происходило до этого.
                </Typography.Text>
            </div>
        );
    })}
</Story>
